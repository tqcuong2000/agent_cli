Design header widget:
- id: `header`
- components:
    - `title`: Static component that displays the title of the application.
    - `quick_action`: Static component that displays the quick action menu.
    - `agent_badge`: Static component that displays the agent badge.
- height: 3 lines
- width: 100%
- layout: horizontal

1. title component:
Show the title of the conversation that generated by agents (backend job)
    - id: `title`
    - content: "Agent CLI"
    - height: 1 line
2. quick_action component:
Show the quick action menu on click
    - id: `quick_action`
    - height: 1 line
    - when user clicks on it, it should open a menu with quick actions (theme, quit) - make it easy to extend for future actions
3. agent_badge component:
Show the current working agent name
    - id: `agent_badge`
    - content: "Main Agent" (For now)
    - height: 1 line
